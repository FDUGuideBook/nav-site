# 写在前面

本项目是复旦相关的各类资源信息的导航站。几年来民间一直有在流传各类开源资料、信息渠道，但是大多比较零散，只能靠自己去搜索，本项目的目标是将这些资源整合起来，方便同学快速获取。

目前主要收录学习相关内容，内容包括但不限于：

- 网站
- GitHub 仓库
- 文档链接
- 网盘链接
- 树洞号

未来可能还会收录：

- 官方网站、公众号列表
- 社团/组织网站、公众号列表
- 指南站点
- 个人博客
- 树洞精华帖（以 PID 形式出现）

## 参与贡献

项目正在发展中，欢迎添加微信（alumls）投稿/提出建议。

对于熟悉 GitHub 工作流程的同学，也可以直接提交 Pull Request / Issue.
